# Simply Legacy 26.03
### Adopted `YY.MONTH.MICRO` versioning
Like Re-Console+, we've once again made a change to the versioning scheme, now using `YY.MONTH.MICRO`, with the first release of the month of March being 26.03, with subsequent patches in the month being amended at the end (`26.03.1`)

### Re-Console+ Parity
- **Legacy Mechanics has finally been added!** - New achievements! Fortune for Melons! Sands in Blast Furnaces!
- **Console Advancement Sounds has been added** - The default sound remains the Java Edition toast sound, but if you'd like to change it, head to the mod options for Console Advancement Sounds.

### Added Fixed Chest Model
Applied by default, thanks to RedRain0o0. This pack fixes issues regarding chest models on the mash-ups.

### Synced Language Files between Simply Legacy and Re-Console+
Comes with a handful of misc. changes and fixes.

### Fixed issue loading the tutorial worlds
Thanks to ZenXArch for patching it!
